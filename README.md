# 3Ds-Max-Modeling-Material-Management-Assistant
3DMAX建模与材质管理助手 是一款专为 3ds Max 用户打造的高效生产力工具。它不仅涵盖了基础的几何体处理与健康检查，更重构了跨场景协作与深度材质管理的工作流，旨在彻底解决建模过程中复杂的材质遗留问题，实现多开软件间的无缝资产流转。

🔥 核心亮点 (Core Highlights)
1. 📋 跨场景协同 (Smart Cross-Scene Workflow)

打破 3ds Max 多开窗口间的数据壁垒，像操作文本一样简单地复制模型。

    一键复制：选中模型点击复制，脚本自动在后台生成临时数据文件。

    无缝粘贴：在另一个 Max 窗口（新项目）中点击粘贴，模型即刻导入。

    微信直发：复制的模型文件保留在剪贴板中，可直接在 微信/QQ 对话框中 Ctrl+V 粘贴，将模型文件发送给同事，或粘贴至文件夹归档。

    ⚠️ 提示 (Note)：为了保证模型数据（如修改器堆栈、材质属性）的完美还原，强烈建议在相同版本的 3ds Max 软件之间 进行复制粘贴操作。

2. 🎨 场景材质管理 (Advanced Material Manager)

专为解决建模过程中的“材质脏乱差”和“遗留问题”而设计。

    精准定位：一键选中使用了特定材质的所有物体，快速排查场景中的材质归属。

    搜索与替换：支持模糊搜索材质名称，配合一键替换功能，将场景中杂乱的材质批量统一为标准材质。

    深度清理：

        自动识别并移除未被引用的“僵尸材质”。

        深度整理 Multi/Sub-Object 多维材质，移除未使用的子材质 ID，修复空缺。

    可视化交互：提供详细的材质预览、参数查看及引用计数，让材质管理一目了然。

✨ 全面功能清单 (All Features)

除了上述两大核心功能外，本助手还集成了以下实用模块：

    🛠️ 几何体工具：支持一键拆分元素为独立物体，或快速合并多个对象。

    🏥 场景健康检查：扫描并修复无效的材质 ID 面，保持模型数据纯净。

    🧹 材质深度去重：三种模式（实例/视觉/同名）检测重复材质，一键合并去重。

    🔄 构件替换：基于 Instance 模式，将源对象批量替换为目标对象。

    📐 智能对齐：基于鼠标拾取目标平面，实现物体的快速法线对齐与吸附。

🚀 安装与使用 (Installation)
环境要求

    3ds Max 版本：2016 +

    系统：Windows (需支持 .NET Framework)

安装步骤

    下载 3dsMax_Modeling_Assistant_v2.1.ms。

    将 .ms 文件直接拖拽到 3ds Max 视口中即可运行。

    脚本启动后会自动停靠在界面右侧。
